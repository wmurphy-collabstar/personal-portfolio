Color Scheme Generator

Check out the Source Code!

This tool was made in mind to recreate a color palette picker based on color and color scheme. The color schemes are 6 of the most common ones, and it works for any color picked.

I used a RESTful API to get color palettes based on hex color value and scheme type. The design is simpler as the focus was on learning how to fetch data from an API.

My main obstacle was figuring out how to use the API, but after studying the documentation, I've found patterns of REST APIs that will help me understand other API documentation as well.

For new features I would like to add a choice for number of colors, and be able to color pick using hex colors or css color keywords to make the form more accessible and customizable.
